SR Comp Mixtape
===============

|Build Status|

A service which plays tracks throughout a competition.

Usage
-----

Install the library using:

.. code:: shell

    pip install git+https://github.com/srobo/srcomp-mixtape


Development
-----------

**Install**:
``pip install -e .``

**Run checks**:
``./script/check``
